### Исправления

**Акции-подарки**: исправлен список при нескольких документах на одного клиента

- Если для одного клиента было создано больше одного документа «Акция Подарок», в списке отображался только первый из них. Теперь показываются все.

**Даты в документах**: исправлен сброс даты после создания из «Маршрутов ТА»

- Если документ «Акция Подарок» или «Возврат» создавался из карточки клиента в разделе «Маршруты ТА», выбранная в календаре дата могла бесследно откатиться на дефолтную («завтра») после возврата к списку клиента — сохранённое значение просто не записывалось в базу. Теперь дата сохраняется надёжно независимо от того, как документ был открыт.

**Надёжность сети**: подсказка про режим полёта теперь работает везде

- Ранее подсказка «попробуйте включить/выключить режим полёта» появлялась только при отправке заявки. Теперь она показывается при любом сбое подключения к серверу — при обновлении статусов заявок, синхронизации справочников, выгрузке трека, обновлении списка точек доставки, возврата.
